# Process Reconciled Suitable Source Albums

Spec for organizing the current reconciled suitable-candidate list from `etc/1-source-files` into `etc/3-organized-files`. The spec executes only destination-unique candidates and blocks candidates that share the same resolved album folder.

Scope: **listed candidates, dated report artifacts, and `etc/3-organized-files` only.** No changes to `etc/1-source-files`, `src/**`, or dependencies.
